نظرة عامة

"يد العون" منصة تبرعات تربط المتبرعين والجمعيات والمستفيدين. تم بناء Back‑End كامل باستخدام ASP.NET Core Web API وربطه بقاعدة بيانات SQL Server مع توفير التوثيق عبر Swagger.

الهدف: منصة تبرعات متكاملة
الدور: Backend Developer
الناتج: API جاهزة للفرونت

التقنيات المستخدمة

ASP.NET Core Web API (.NET 10)
Entity Framework Core
SQL Server
JWT Authentication
BCrypt Password Hashing
Swagger UI & OpenAPI

الأمان والتوثيق

  • تشفير كلمات المرور باستخدام BCrypt.
  • JWT Token للتصديق على الطلبات المحمية.
  • Swagger لتوثيق كامل الـ API وتجربتها مباشرة.
Authorization: Bearer <TOKEN>

التحديات والحلول

مشكلة Swagger Invalid
الحل:
  • ضبط إعدادات Swagger UI وتثبيت المسار.
  • تقديم Swagger على الجذر لسهولة الوصول.
500 Error في إنشاء تبرع
الحل:
  • حل مشكلة cycle في JSON باستخدام ReferenceHandler.IgnoreCycles.
  • إعادة هيكلة الاستجابة لتفادي التكرار.
Mismatch في أسماء الجداول
الحل:
  • تثبيت أسماء الجداول يدويًا في EF Core.
modelBuilder.Entity<Donor>().ToTable("Donor")
